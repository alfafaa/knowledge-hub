# Review And Lifecycle Policy

## Review Cadence

- operational and security docs: 90 days
- standards and product docs: 180 days
- stable reference docs: 365 days

## Statuses

- `draft`
- `review`
- `approved`
- `active`
- `deprecated`
- `archived`

## Rule

Every publishable document must have:

- a named owner
- a `last_reviewed` date
- a review cycle

# Publishable Guide Template

Use this as the starting point for guides that may be mirrored and published to `public-site`, `internal-site`, or `engineering-site`.

Recommended frontmatter:

```yaml
---
title: Guide Title
type: guide
status: approved
owner: team-name
publish: true
audience: internal
visibility: internal
publish_targets:
  - internal-site
---
```

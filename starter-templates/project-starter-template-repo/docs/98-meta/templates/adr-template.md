# ADR Template

Copy this into `docs/03-decisions/adr/adr-<number>-<slug>.md` when a meaningful architecture or engineering decision is made.

Recommended frontmatter for a real ADR:

```yaml
---
title: Use Event Bus For Cross-Service Notifications
type: adr
status: approved
owner: team-platform
publish: true
audience: engineering
visibility: internal
publish_targets:
  - engineering-site
---
```

## Decision

State the decision in one sentence.

## Context

Explain the problem, constraints, and forces that led to the decision.

## Options Considered

- option 1
- option 2
- option 3

## Chosen Option

Explain which option was chosen and why.

## Consequences

- positive outcomes
- trade-offs
- risks introduced

## Follow-Up

List work that should happen because of this decision.
